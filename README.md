# Žaidimų Alėja

Svetainė, skirta stalo žaidimų peržiūrai, paieškai ir filtravimui.

## Aprašymas
Projektas leidžia vartotojams:
- Peržiūrėti įvairius stalo žaidimus
- Filtruoti pagal žanrą ir žaidėjų skaičių
- Ieškoti žaidimų pagal pavadinimą
- Registruotis ir prisijungti
- Peržiūrėti renginius ir palikti atsiliepimus

## Funkcionalumas
- Žaidimų paieška
- Filtravimas pagal kategorijas
- Prisijungimo ir registracijos formos
- Tamsus / šviesus režimas
- Atsiliepimų sistema
- Renginių puslapis

## Naudotos technologijos
- HTML
- CSS
- JavaScript
Duomenų bazė nėra prijungta
